Open the project in VS Code or Rider. 
Build and run the Game1.cs file.

GAME CONTROLS

Movement:
W or Up Arrow       - Move forward
S or Down Arrow     - Move backward
A or Left Arrow     - Move left
D or Right Arrow    - Move right

Other Controls:
Space   - Restart game (on Game Over or Victory screen)
Escape  - Quit game

HOW TO PLAY

Objective:
Guide the duck across the road while avoiding vehicles.
Reach 50 points to win.

Gameplay:
- You control a yellow duck starting at the bottom
- Move forward to increase your score
- Avoid vehicles moving across lanes
- Each forward move gives +1 point
- Reaching the top respawns you at the bottom

End Conditions:
- Win: Reach 50 points → Victory screen
- Lose: Hit by a vehicle → Game Over screen

End Screens:
- Show final score
- Show restart option (Space)
- Show quit option (Escape)
- Game Over screen also shows high score

TIPS

- You cannot interrupt a move once it starts
- Watch traffic patterns before moving
- You can move backward to avoid danger
- High score stays during the session


Enjoy the game!
